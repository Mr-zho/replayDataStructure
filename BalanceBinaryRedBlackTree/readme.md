红黑树性质：
    1.结点是RED或者BLACK
    2.根结点是BLACK
    3.叶子结点(外部结点，空结点)都是BLACK
    4.RED结点的子结点都是黑色
        4.1. RED结点的parent都是BLACK
        4.2 从根结点到叶子结点的所有路径上不能有2个连续的RED结点
    5.从任一结点到叶子结点的所有路径都包含相同数目的BLACK结点


B树性质
    [
        1.一个结点可以存储超过2个元素, 可以拥有超过2个子结点。
        2.拥有二叉搜索树的一些性质
        3.平衡, 每个结点的所有子树高度一致
    ]
